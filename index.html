Project 2

Hangman

This project is to individually implement a popular rudimentary game in JavaScript - "hang man". As support, there will be a code and a structure that you can use when solving the task. Hang Man, is a game where the word is random. The player then guesses which letters the word contains, for each incorrect answer a part of the "hang old man" picture is drawn. If the player guesses wrong enough times and the old man has been "hanged", the player has lost. However, if the player manages to guess all the letters, the player wins.
